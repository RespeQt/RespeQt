LINUX:
======
Install Qt5 libraries and build tools then type the following in a terminal:

sudo apt install qtbase5-dev qt5-default qtcreator libqt5serialport5-dev libqt5svg5-dev libcups2-dev
qmake
make


Mac OS X:
=========
Install Qt5 libraries and build tools then type the following in a terminal:

qmake -spec macx-g++
make
